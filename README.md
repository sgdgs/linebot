# **爬山小助手**
### **在登山前查詢天氣狀況**
協助山友了解登山前的天氣狀況，可以提早做好準備:)

![image](https://github.com/sgdgs/linebot/assets/143071559/513a10df-c553-4af3-9eea-2db8131a650d)

## 使用說明
### 輸入臺灣內的山名
每次使用一個單詞搜尋 ex:玉山、巴巴山... <br>
便可得知該山近期氣溫及舒適度<br>
如使用相關字詞，則會列出10筆關資料
